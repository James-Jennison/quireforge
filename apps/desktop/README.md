# QuireForge Desktop

Status: Milestone 3 scaffold implemented and verified locally. This package
establishes the native shell and a small typed Tauri command boundary. Codex
runtime, authentication, project attachment, persistence, terminal, Git, and
integration features are not implemented here yet.

Run package checks from the repository root with `pnpm validate`. Start the
native development window with `pnpm desktop:dev` after installing the Linux
packages documented in [`docs/BUILDING.md`](../../docs/BUILDING.md).
